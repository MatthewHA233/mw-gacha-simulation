# 现代战舰抽奖模拟器 - 版本框架清单

本文档记录了项目的完整版本历史，包含每个版本对应的 git 提交编号，方便后续 AI 根据清单逐个详细说明版本更新内容。

---

## 版本号规则

- **MAJOR**: 重大架构变更（如 1.0.0 的架构重构）
- **MINOR**: 新的整体组件化系统上线（旗舰宝箱、机密货物）
- **PATCH**: 功能优化、Bug修复、UI改进、延伸功能（如无人机补给）
